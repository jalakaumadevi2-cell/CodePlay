# CodePlay - JavaScript Playground

**CodePlay** is an interactive online playground for writing, executing, and testing JavaScript code directly in the browser. It is designed for learning, experimenting, and debugging small code snippets without any setup.

## Features
- Interactive code editor for JavaScript
- Run code and display output instantly
- Clear output console for results or errors
- Responsive and minimal interface
- Stores recent code snippets using `localStorage`

## Optional Enhancements
- Syntax highlighting (can be added with libraries like CodeMirror or Prism)
- Shareable code snippets via export
- Light/Dark theme customization
- Support for HTML & CSS for full web demos

## Usage
1. Open `index.html` in your browser.
2. Write JavaScript code in the editor.
3. Click "Run Code" to execute.
4. Click "Clear Output" to reset the console.
